# Server-Side JavaScript

## Overview
Introduction to Node.js as a backend alternative to PHP, Java, or Python, covering core modules, event-driven architecture, and web application development with Express.

## Learning Objectives
- Use Node.js to build backend applications
- Understand the Node core and event-driven model
- Manage modules with CommonJS (`require`, `exports`)
- Build web applications with Express and NPM

## Course Content
- Introduction to Node.js
- The Node core
- Modules, exports, and imports (`require`)
- Express framework
- NPM (package management)
